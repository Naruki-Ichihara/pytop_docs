Cantilever problem

In this example, a linear (2D plane stress) elasticity problem is considered. First, we show forward definition of the problem using FEniCS in pytop. Then, we describe the how to build the optimization problem using pytop's features.

Forward Problem​

Governing Equation​

The equation for linear elastic deformation problem can be written as

−∇⋅σ=f in Ω-\nabla \cdot \sigma = {\bm f} \ {\rm in}\ \Omega−∇⋅σ=f in Ω σ=λtr(ε)I+2με\sigma = \lambda {\rm tr} (\varepsilon)I+2\mu \varepsilonσ=λtr(ε)I+2με ε=12(∇u+(∇u)T)\varepsilon = \frac{1}{2}\left(\nabla {\bm u}+(\nabla {\bm u})^{\rm T} \right)ε=21​(∇u+(∇u)T)

where σ\sigmaσ is the stress tensor, f{\bm f}f is a body force, and u{\bm u}u is the deformation vector. The strain ε\varepsilonε can be calculated from the u{\bm u}u. The material property is described as Lame's constants, λ\lambdaλ and μ\muμ. All variables are defined in closed body Ω\OmegaΩ.

Variational Form​

The variational (or weak) form of abovementioned equation is obtained by the inner product of the equation and test function v{\bm v}v. We get

∫Ωσ:∇v dV=∫∂Ωf⋅v dV+∫∂Ω(σ⋅n)⋅v dS.\int_\Omega\sigma : \nabla {\bm v}\ dV = \int_{\partial \Omega}{\bm f}\cdot{\bm v}\ dV + \int_{\partial \Omega}\left( \sigma \cdot {\bm n}\right)\cdot {\bm v}\ dS.∫Ω​σ:∇v dV=∫∂Ω​f⋅v dV+∫∂Ω​(σ⋅n)⋅v dS.

Here, (σ⋅n)=t\left( \sigma \cdot {\bm n}\right)={\bm t}(σ⋅n)=t is a traction force on the Dirichlet bondary ∂ΩD{\partial \Omega_D}∂ΩD​. This form can summarize as: find u∈V{\bm u}\in Vu∈V that satified

a(u,v)=L(v)a({\bm u},{\bm v})=L({\bm v})a(u,v)=L(v) a(u,v)=∫Ωσ:∇v dVa({\bm u},{\bm v})=\int_\Omega\sigma : \nabla {\bm v}\ dVa(u,v)=∫Ω​σ:∇v dV L(v)=∫∂Ωf⋅v dV+∫∂Ω(σ⋅n)⋅v dS.L({\bm v})=\int_{\partial \Omega}{\bm f}\cdot{\bm v}\ dV + \int_{\partial \Omega}\left( \sigma \cdot {\bm n}\right)\cdot {\bm v}\ dS.L(v)=∫∂Ω​f⋅v dV+∫∂Ω​(σ⋅n)⋅v dS.

Because of symmetrity of the σ\sigmaσ, the bilinear form a(u,v)a({\bm u},{\bm v})a(u,v) can be replaced into elastic potential:

a(u,v)=∫Ωσ(u):ε(v) dVa({\bm u},{\bm v})=\int_\Omega\sigma({\bm u}) : \varepsilon({\bm v})\ dVa(u,v)=∫Ω​σ(u):ε(v) dV
